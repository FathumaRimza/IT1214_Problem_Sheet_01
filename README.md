# Java Practice Projects

## Description
A collection of small Java practice programs to improve problem-solving and programming skills.
Includes basic exercises covering loops, arrays, OOP concepts, and data structures.

## Tech / Concepts Used
- Java 8
- Object-Oriented Programming (OOP)
- Loops, Conditional Statements
- Arrays and Strings



## How to Run
1. Install Java JDK (version 8 or above)
2. Open project folder in any IDE (Eclipse, IntelliJ, VS Code)
3. Compile using `javac FileName.java`
4. Run using `java FileName`
